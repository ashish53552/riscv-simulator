# RISC-V-Simulator

A Simulator for RISC-V Instruction Set Architecture (ISA) built in Python3.
      
How to build
------------
For building:
	$cd src
	$make

For cleaning the project:
	$cd src
	$make clean


How to execute
--------------
./myRISCVSim test/simple_add.mc
